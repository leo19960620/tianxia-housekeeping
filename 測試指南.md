### 第三步:啟動React Native App

**重要**: 保持後端伺服器持續運行,開啟新的終端機視窗

1. **確認Android Studio已安裝並設定好模擬器**
   - 如未安裝,請先下載 [Android Studio](https://developer.android.com/studio)
   - 安裝後開啟AVD Manager建立一個Android模擬器

2. **進入App目錄**
   ```bash
   cd /Users/leo/tianxia-housekeeping/TianxiaHousekeeping
   ```

3. **啟動Metro Bundler**
   ```bash
   npm start
   ```

4. **在新的終端機視窗執行Android App**
   ```bash
   cd /Users/leo/tianxia-housekeeping/TianxiaHousekeeping
   npm run android
   ```
   或
   ```bash
   npx react-native run-android
   ```

**預期流程**:
- Gradle會開始編譯Android專案(首次需要較長時間)
- 編譯完成後會自動啟動Android模擬器
- App安裝並自動開啟

## 🐛 可能遇到的問題

### 問題1: Android編譯錯誤
**解決方法**:
```bash
cd android
./gradlew clean
cd ..
npm run android
```

### 問題2: Metro Bundler錯誤
**解決方法**:
```bash
npm start -- --reset-cache
```

### 問題3: 無法連接API
**檢查項目**:
1. 後端伺服器是否正在運行?
2. 檢查 `src/config/constants.js` 中的 `API_CONFIG.BASE_URL`
3. 如果使用模擬器測試,確認已連接到localhost

**Android模擬器連接本機API**:
編輯 `TianxiaHousekeeping/src/config/constants.js`:
```javascript
export const API_CONFIG = {
  // Android模擬器使用這個
  BASE_URL: 'http://10.0.2.2:3000/api',
  TIMEOUT: 10000,
};
```

### 問題4: Android Studio未安裝
請先安裝Android Studio:
1. 下載: https://developer.android.com/studio
2. 安裝完成後設定環境變數(參考App的README.md)
