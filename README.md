# 天下客房部庫存管理系統

一個現代化的飯店客房部交接管理應用程式，使用 React + Vite + Firebase 開發。

## 功能特色

- ✅ **公告管理** - 顯示當前重要公告與清潔注意事項
- ✅ **交接紀錄** - 記錄早/中/晚班的交接事項
- ✅ **備品管理** - 追蹤收放備品紀錄
- ✅ **臭氧紀錄** - 登記臭氧使用房號
- ✅ **交班事項** - 記錄與查詢重要交班資訊
- ✅ **響應式設計** - 支援平板與手機裝置

## 技術棧

- **前端**: React 18 + Vite
- **路由**: React Router DOM
- **資料庫**: Firebase Firestore
- **樣式**: CSS Modules

## 快速開始

### 1. 安裝依賴
\`\`\`bash
npm install
\`\`\`

### 2. 設定 Firebase
請參考 [Firebase 設定指引](../../../brain/df7a59cd-2f9e-4740-98d8-ab698a1b231e/firebase_setup.md)

1. 複製環境變數範例檔案：
\`\`\`bash
copy .env.example .env
\`\`\`

2. 編輯 `.env` 檔案，填入您的 Firebase 專案設定

### 3. 啟動開發伺服器
\`\`\`bash
npm run dev
\`\`\`

應用程式將在 `http://localhost:5173` 運行

## 資料庫結構

### Collections
- **announcements** - 公告資料
- **handovers** - 交接紀錄
- **inventory_records** - 收放備品紀錄
- **ozone_records** - 臭氧使用紀錄
- **handover_items** - 交班事項
- **items** - 備品清單

詳細資料結構請參考 `implementation_plan.md`
