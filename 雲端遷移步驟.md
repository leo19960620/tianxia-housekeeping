## 第四步：部署後端到 Render

### 1. 註冊與建立服務
1. 前往 [Render Dashboard](https://dashboard.render.com/)
2. 點擊 **New +** → **Web Service**
3. 選擇 **Build and deploy from a Git repository**
4. 連結您的 GitHub 帳號，並選擇 `tianxia-housekeeping` 專案
   - 如果沒看到，點擊 "Configure account" 授權 Render 存取該儲存庫

### 2. 設定服務參數
填寫以下資訊：
- **Name**: `tianxia-housekeeping-api`
- **Region**: 選擇 `Singapore (Southeast Asia)` (離台灣近)
- **Branch**: `main`
- **Root Directory**: `backend` (非常重要！一定要填)
- **Runtime**: `Node`
- **Build Command**: `npm install`
- **Start Command**: `npm start`
- **Instance Type**: 選擇 **Free** (免費版) 或 **Starter** (USD 7/月，推薦，不休眠)

### 3. 設定環境變數 (Environment Variables)
請複製以下內容，並在 Render 的 **Environment** 區塊逐一新增：

| Key | Value |
|-----|-------|
| `DB_HOST` | `aws-1-ap-northeast-1.pooler.supabase.com` |
| `DB_PORT` | `5432` |
| `DB_USER` | `postgres.yyjcuhacwlwvhesvzhak` |
| `DB_PASSWORD` | `CGd5rHzkX@TTbgg` |
| `DB_NAME` | `postgres` |
| `NODE_ENV` | `production` |
| `PORT` | `3001` |
| `JWT_SECRET` | `tianxia_housekeeping_secret_key_2026` |
| `JWT_EXPIRES_IN` | `7d` |

> 💡 **提示**：您可以使用 Render 的 "Add from .env" 功能，將上面的內容存成文字檔貼上，或手動一筆一筆新增。

### 4. 開始部署
1. 點擊頁面底部的 **Create Web Service**
2. 畫面會跳轉到部署日誌 (Logs)
3. 等待約 2-5 分鐘，直到看到 `Server is running on port 3001` 或 `Your service is live`

### 5. 取得後端網址
部署成功後，左上角標題下方會有一個網址，例如：
`https://tianxia-housekeeping-api.onrender.com`
**請複製這個網址**，我們下一步要更新前端。

### 5. 測試 API
在瀏覽器開啟：
```
https://tianxia-housekeeping-api.onrender.com/health
```

應該看到 JSON 回應表示成功
