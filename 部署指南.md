# 天下房務管理系統 - 部署與更新指南

本文件說明如何維護與更新系統的三個平台版本。

## 📱 Android (APK)

**打包更新版本：**

1. 確保 `android/app/build.gradle` 中的版本號 (`versionCode`, `versionName`) 已更新。
2. 執行打包指令：
   ```bash
   cd TianxiaHousekeeping/android
   ./gradlew assembleRelease
   ```
3. 打包完成後，APK 位於：
   `TianxiaHousekeeping/android/app/build/outputs/apk/release/app-release.apk`
4. 將 APK 複製到網站公開目錄：
   ```bash
  cp android/app/build/outputs/apk/release/app-release.apk ../public/
   ```
5. 重新部署網站以更新下載連結。

---

## 🍎 iOS (Xcode Direct Install)

**安裝開發版本：**

1. 連接 iPhone 至 Mac。
2. 開啟 `TianxiaHousekeeping/ios/TianxiaHousekeeping.xcworkspace`。
3. 在 Xcode 中選擇您的 iPhone 作為目標裝置。
4. 按下 `Cmd + R` 進行編譯與安裝。
5. **注意**：免費開發者帳號簽署的 App 有效期為 7 天，過期後需重新執行此步驟。

---

## 🌐 Web (Cloudflare Pages)

**部署新版本：**

1. 建置生產版本：
   ```bash
   npm run build
   ```
2. 部署到 Cloudflare Pages（若設定為 Git 自動部署則推送代碼即可）。
   若需手動部署：
   ```bash
   npx wrangler pages deploy dist --project-name=tianxia-housekeeping
   ```

## ⚙️ 環境配置

- **Web**: `.env.production` 控制生產環境變數。
- **React Native**: `src/config/constants.js` 內建邏輯根據 `__DEV__` 變數自動切換 API 網址。
  - Debug 模式：使用 Localhost (10.0.2.2)
  - Release 模式：使用 https://housekeeping.leolnfsystem.dpdns.org/api

## ⚠️ 常見問題

1. **Android 無法安裝**：請確認已解除安裝舊的 Debug 版本，且手機已允許安裝來自此來源的應用程式。
2. **iOS App 無法開啟**：請確認在 iPhone 設定 > 一般 > VPN 與裝置管理 中已信任開發者憑證。
3. **API 連線失敗**：請確認 Cloudflare Tunnel 正常運作，且 API 伺服器 (Backend) 正在執行。
